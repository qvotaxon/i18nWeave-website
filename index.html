Getting Started

Show Instructions ▾

Step 1: Open Command Palette

Open the command palette by pressing Ctrl+Shift+P (Windows/Linux) or Cmd+Shift+P (Mac).

Step 2: Run Configure Command

In the command palette, type Configure i18nWeave and select it.

Ctrl+Shift+P (Windows/Linux) or Cmd+Shift+P (Mac)

Step 3: Single Project Configuration

If you are not using a Mono-repo, select Single Project when prompted. Note that monorepos are not supported yet.

Step 4: Choose Framework

You will be prompted to choose among several frameworks. Since none are implemented yet, select Other (custom).

Step 5: Select Translation Files Location

Choose the location of your translation files. A folder browser will appear for you to select the appropriate folder.

Step 6: Select Code Files Location

Next, choose the location of your code files. A folder browser will appear, and you will be allowed to select multiple folders.

Step 7: Enter Default Language

You will need to enter your default language. This can be in formats like nl, en, nl-NL, or en-GB.

Step 8: Enter Supported Languages

Enter all the supported languages you want to include, separated by commas. Like nl, en, fr.

Step 9: Enter Translation Namespaces

Finally, enter the namespaces of your translations, such as common and navigation.

You have now succesfully configured I18nWeave. Happy weaving! üåç
